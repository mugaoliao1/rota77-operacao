🔑 GitHub Personal Access Token
Necessário para publicar escala, boletim e radar
✅
Como gerar o token
  1. Acesse github.com/settings/tokens (Settings → Developer settings → Personal access tokens → Tokens (classic))
  2. Clique em Generate new token (classic)
  3. Dê um nome (ex.: rota77-operacao) e defina a expiração
  4. Marque o escopo repo (acesso completo a repositórios privados e públicos)
  5. Clique em Generate token e copie o valor gerado — ele começa com ghp_
  6. Cole o token no campo acima e clique em Salvar token
⚠️ Atenção: use Tokens (classic), não Fine-grained tokens. O token é salvo apenas no localStorage deste navegador e nunca enviado a servidores externos.
